# Landing Page SmartTech X100

## Descripción
Este proyecto es una landing page adaptativa para el producto SmartTech X100.  
El objetivo fue aplicar todo lo aprendido sobre **diseño responsivo**, media queries y estrategias mobile first y desktop first, asegurando que la página se vea bien en cualquier dispositivo: móviles, tablets y escritorio.

## Estrategia utilizada
Se utilizó **Mobile First**, diseñando primero para pantallas pequeñas y luego ajustando estilos para tablets y escritorios mediante media queries. Esto asegura que el contenido principal siempre sea accesible y legible, incluso en dispositivos con menor resolución.

## Implementación técnica
- **Media queries:** se aplicaron para cambiar tamaños de fuentes, disposición de elementos, menú, galería y formulario según el ancho del viewport.
- **Breakpoints:** se definieron en 480px (móvil pequeño), 768px (tablet) y 1200px (escritorio grande).
- **Galería adaptativa:** se utiliza `grid` y cambia de 1 a 3 columnas según el ancho de pantalla.
- **Formulario flexible:** se reorganiza verticalmente en móviles y horizontalmente en escritorio.
- **Estilos de impresión:** se oculta el menú y se ajusta el contenido para imprimir.

## Cómo usar
1. Abrir `index.html` en cualquier navegador moderno.
2. Redimensionar la ventana para ver cómo cambian los estilos automáticamente.
3. Explorar las secciones de características, galería y contacto.

## Resultado
Se logró un diseño limpio, atractivo y totalmente funcional en diferentes dispositivos, cumpliendo con los requerimientos del proyecto.
